# Photography Sidebar Website

A responsive photography-themed landing page built with HTML and CSS. It features a full-screen background image, dark overlay, animated sidebar navigation, social media icons, and category cards for photography services.

## Features

- Full-screen photography background
- CSS-only sidebar menu
- Hamburger and close button controls
- Smooth hover and slide animations
- Photography category cards
- Responsive layout for smaller screens
- Font Awesome icons
- Google Fonts integration

## Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts

## Project Structure

```text
Mini CSS Project/
+-- index.html
+-- style.css
+-- photo.jpg
+-- README.md
```

## How to Use

Open `index.html` in your browser to view the project.
